# Multiple user chatroom
A little practice for React.js and GraphQL.

## Skill
1. React.js with class component
2. hook
3. graphQL with Apollo server and Apollo client
4. use fake DB

## How to start?
### start server first
```
cd .\server\
yarn
yarn start
```

### start client
```
cd .\client\
yarn
yarn start
```
